# Yosys synthesis reports

Generated by:

```bash
cd sim
make synth
# or: bash synth.sh
```

## Flow

| Step | Tool | Purpose |
|------|------|---------|
| Read RTL | Yosys `read_verilog -sv` | All `rtl/*.sv` files |
| Elaborate | `hierarchy -top apb_slave_top` | Default `USE_APB_FSM=0` |
| Optimize | `proc; opt; fsm; memory; opt` | Combinational + sequential opt |
| Map | `techmap; opt` | Generic gate mapping |
| Report | `stat; check` | Cell counts, sanity check |

## Outputs

| File | Description |
|------|-------------|
| `synth_report.txt` | Console log with `stat` summary |
| `yosys.log` | Full Yosys log |
| `apb_slave_top_syn.v` | Structural netlist (generic cells) |
| `apb_slave_top.json` | JSON netlist for downstream tools |

## CI

GitHub Actions job **wave-and-synth** runs `make synth` on Ubuntu and uploads this folder as an artifact.

After push, download **wave-synth-reports** from the Actions tab if you need fresh logs locally.

## Expected resource (USE_APB_FSM=0)

| Resource | Approx. |
|----------|---------|
| Register storage | 20 × 32-bit = **640 flip-flops** |
| Wait-state counter | 3-bit counter + compare |
| Address decode | Combinational one-hot + window check |
| Read mux | 20:1 × 32-bit multiplexing |

Exact `stat` cell counts appear in `synth_report.txt` after a successful run.
